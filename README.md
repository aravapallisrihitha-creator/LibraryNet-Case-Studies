# LibraryNet – Intelligent Digital Library & Knowledge Retrieval System

## Project Description
This repository contains DSA-based case studies implemented for the LibraryNet project domain. The system demonstrates efficient library management and knowledge retrieval using advanced data structures and graph algorithms.

The project includes:
- AVL Tree Based Library Catalog Management
- B-Tree Based Digital Library Indexing
- Graph-Based Book Recommendation and Citation Network

---

# CO1 – AVL Tree Based Library Catalog Management

## Description
This case study implements AVL Trees for efficient storage and management of library books while maintaining a balanced search structure.

## Features
- Insert Book
- Search Book
- Display Books
- Update Book
- Delete Book
- AVL Balancing

## Concepts Used
- AVL Trees
- Binary Search Trees
- Tree Traversal
- Rotations

---
