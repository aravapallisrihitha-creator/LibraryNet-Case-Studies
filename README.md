# 📚 LibraryNet – Intelligent Digital Library & Knowledge Retrieval System

## Overview

LibraryNet – Intelligent Digital Library & Knowledge Retrieval System is a collection of six Data Structures and Algorithms (DSA) case studies developed as part of the academic curriculum. Each case study demonstrates the application of a different data structure or algorithm in solving real-world problems related to digital library management. The project focuses on efficient book management, searching, sorting, routing, recommendation, and data retrieval techniques.

---

## Objectives

* Develop efficient library management solutions using DSA concepts.
* Improve searching, sorting, and routing performance.
* Demonstrate practical applications of data structures and algorithms.
* Enhance digital library services through optimized implementations.

---

## Technologies Used

* **Programming Language:** Java
* **IDE:** Visual Studio Code
* **Data Structures:** AVL Tree, Graph, Arrays, Dynamic Programming
* **Algorithms:** AVL Rotations, Bidirectional Dijkstra's Algorithm, Merge Sort, Longest Common Subsequence (LCS)

# Case Studies

## CO1 – AVL Tree Based Library Catalog Management

### Description

Implements an AVL Tree to efficiently manage library book records. The system supports insertion, searching, updating, deletion, and balanced storage of books.

### Operations

* Add Book
* Search Book
* Update Book
* Delete Book
* Display Book Catalog

## CO4 – Library Branch Book Delivery Route Optimization

### Description

Uses Bidirectional Dijkstra's Algorithm to determine the shortest route for delivering books between library branches.

### Operations

* Add Library Branches
* Add Routes
* Find Shortest Path
* Display Delivery Route
* Display Minimum Distance

---

## CO5 – Efficient Library Catalog Sorting

### Description

Uses Merge Sort to efficiently sort library books based on Book ID for faster retrieval and organized catalog management.

### Operations

* Add Book IDs
* Sort Book Records
* Merge Sorted Lists
* Display Sorted Catalog

---

## CO6 – Intelligent Book Title Matching

### Description

Uses the Longest Common Subsequence (LCS) algorithm to identify the closest matching book title when users enter incomplete or misspelled search queries.

### Operations

* Store Book Titles
* Search Book
* Compare Titles
* Calculate Similarity
* Recommend Closest Match
