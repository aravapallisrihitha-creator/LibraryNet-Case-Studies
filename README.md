# AVL Tree Based Library Catalog Management

## Project Description
This project implements an AVL Tree for managing library book records efficiently. The system allows insertion, searching, and displaying of books while maintaining a balanced tree structure.

## Features
- Insert Book Records
- Search Books using Book ID
- Display Books in Sorted Order
- Automatic AVL Tree Balancing

## Technologies Used
- Java
- AVL Tree Data Structure

## Concepts Used
- Binary Search Tree
- AVL Rotations
- Tree Traversal
- Balanced Search Structures

## How to Run

Compile:
javac AVLTreeLibrary.java

Run:
java AVLTreeLibrary

## Sample Operations
1. Insert Book
2. Search Book
3. Display Books
4. Exit

## Application
This system can be used in digital libraries for fast and efficient book management.
